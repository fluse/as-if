# Jukebox

## Installation

run following command

```
> cd scripts/ && sh install.sh
```


### Raspberry Pi

```
> amixer controls
```

```
> alsamixer
```


store sound settings

```
> sudo alsactl store
```


#### Install Audio

```
> apt-get install libasound2-dev
```

#### Connect to Pi
```
> ssh pi@192.168.100.79
```
